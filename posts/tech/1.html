快速部署gpt到网站

通过网络代理部署gpt^1。

准备工作

  1. 访问openai官网利用Google邮箱与国外手机收验证码验证后获得API_KEY
  2. 利用宝塔面板初始化网站应用,域名可直接填公网ip

clash代理

  1. 下载clash(ps:最好clone而非安装包解压,否则可能需用chmod调整文件权限,当然这在log文件中会提示你)
$ git clone https://github.com/wanhebin/clash-for-linux.git
  1. 编辑start.sh脚本文件，修改变量URL的值,此值为Clash订阅地址
$ cd clash-for-linux
$ vim start.sh
  1. 启动程序
$ sh start.sh
Clash订阅地址可访问！                                      [  OK  ]
配置文件config.yaml下载成功！                              [  OK  ]
服务启动成功！                                             [  OK  ]

Clash Dashboard 访问地址：http://IP:9090/ui
Secret：xxxxxxxxxxxxx

请执行以下命令加载环境变量: source /etc/profile.d/clash.sh

请执行以下命令开启系统代理: proxy_on

若要临时关闭系统代理，请执行: proxy_off
  1. 检查服务端口,环境变量
$ netstat -tln | grep -E '9090|789.'

$ env | grep -E 'http_proxy|https_proxy'
http_proxy=http://127.0.0.1:7890
https_proxy=http://127.0.0.1:7890
  1. 登录Clash Dashboard选择代理地区(日本优化)即可,登录需用到3.中Secret值

​ ps:Clash Dashboard中代理列表是空白,在项目github中的issues里已被解决,解决方法为将2.步中所用Clash订阅地址通过在线转换平台转换为短链接填入启动脚本

php代码

​ 使用chatgpt自己生成的代码,略作相应修改即可

调用部分:

<?php

function chatWithGPT($message)
{
    $messagee=[["role"=> "user", "content"=> $message]];
    $url = 'https://api.openai.com/v1/chat/completions';
    
    $data = [
        'model' => 'gpt-3.5-turbo',
        'messages' => $messagee
    ];

    $headers = array('Authorization:Bearer sk-Q6cZ5rzXXXXXXXX', 'Content-Type: application/json');

    $ch = curl_init();

    curl_setopt($ch, CURLOPT_URL, $url);
    curl_setopt($ch, CURLOPT_RETURNTRANSFER, 1);
    // curl_setopt($ch, CURLOPT_SSL_VERIFYPEER, false);
    // curl_setopt($ch, CURLOPT_SSL_VERIFYHOST, false);
    curl_setopt($ch, CURLOPT_PROXY, '127.0.0.1:7890');
    // curl_setopt($ch, CURLOPT_PROXYTYPE, CURLPROXY_SOCKS5);
    curl_setopt($ch, CURLOPT_POST, 1);
    curl_setopt($ch, CURLOPT_HTTPHEADER, $headers);
    // curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode($data));
    curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode($data));
    curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);

    $response = curl_exec($ch);
    curl_close($ch);

    $answer = json_decode($response, true)['choices'][0]["message"]["content"];

    return $answer ;
}

// $reply = chatWithGPT($input);
$reply = chatWithGPT( $_POST["comment"]);

echo "<div>$reply</div>";
?>
